How Payroll Works

  1. Add Workers — Enter each team member's email, role, salary, and MoMo details above.
  2. Link Paystack — Click "Link Paystack" to create a transfer recipient for each worker.
  3. Generate Payroll — At the start of each month, click "Generate Payroll" to create pending records.
  4. Run Payroll — Click "Run Payroll" to send all pending transfers from your Paystack balance.
  5. Pause/Resume — Toggle any worker off if funds are tight. They won't be included in next generation.
